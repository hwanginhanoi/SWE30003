# SWE30003 - Backend

## Smart Online Parking System

### Requirements
- Node.js
- Docker

### 1. Clone 
```shell
git clone https://github.com/hwanginhanoi/SWE30003.git
```

### 2. Install
```
npm install
primsa migrate dev --name init
docker-compose up -d
npm run start
```

The backend server will be running on `http://localhost:3000`
